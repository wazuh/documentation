#. Add NodeJS repository:

    .. code-block:: console

      # curl -sL https://deb.nodesource.com/setup_10.x | bash -

#. Install NodeJS:

    .. code-block:: console

      # apt-get install nodejs

#. Install the Wazuh API. It will update NodeJS if necessary:

    .. code-block:: console

      # apt-get install wazuh-api
