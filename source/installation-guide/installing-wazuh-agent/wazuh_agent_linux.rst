Install Wazuh agent on Linux
============================

The Wazuh agent can be installed in the most of Linux Distribution. It's possible to use DEB packages or RPM packages depending on the target Operative System flavor.


+-------------------------------+------------------------------------------------------+
| Operative system              | Version                                              |
+===============================+======================================================+
| Amazon Linux                  | :doc:`1 or greater <wazuh_agent_linux_rpm>`          |
+-------------------------------+------------------------------------------------------+
| CentOS                        | :doc:`6 or greater <wazuh_agent_linux_rpm>`          |
+                               +------------------------------------------------------+
|                               | :doc:`5 <wazuh_agent_linux_rpm_5>`                   |
+-------------------------------+------------------------------------------------------+
| Red Hat Enterprise Linux      | :doc:`6 or greater <wazuh_agent_linux_rpm>`          |
+                               +------------------------------------------------------+
|                               | :doc:`5 <wazuh_agent_linux_rpm_5>`                   |
+-------------------------------+------------------------------------------------------+
| Oracle Linux                  | :doc:`6 or greater <wazuh_agent_linux_rpm>`          |
+                               +------------------------------------------------------+
|                               | :doc:`5 <wazuh_agent_linux_rpm_5>`                   |
+-------------------------------+------------------------------------------------------+
| Debian                        | :doc:`7 or greater <wazuh_agent_linux_deb>`          |
+-------------------------------+------------------------------------------------------+
| Ubuntu                        | :doc:`12.10 or greater <wazuh_agent_linux_deb>`      |
+-------------------------------+------------------------------------------------------+
| Fedora                        | :doc:`22 or greater <wazuh_agent_linux_fedora>`      |
+-------------------------------+------------------------------------------------------+
| OpenSUSE                      | :doc:`42 or greater <wazuh_agent_linux_rpm_suse_12>` |
+-------------------------------+------------------------------------------------------+
| SUSE                          | :doc:`12 <wazuh_agent_linux_rpm_suse_12>`            |
+                               +------------------------------------------------------+
|                               | :doc:`11 <wazuh_agent_linux_rpm_suse_11>`            |
+-------------------------------+------------------------------------------------------+
